# FitnessFoodComparator

#### Fullstack Challenge By Coodesh

--- 
App para mostrar e comparar dados sobre alimentos retirados do [WorldFoodFacts](https://world.openfoodfacts.org/).

### Techs
- NodeJs
- Express
- Cheerio
- Axios
- MongoDB
- Angular

### Como utilizar
#### Requisitos
- Node
- Angular CLI

##### Suba a API
`node API/index.js`

##### Rode o app angular
`ng serve`
